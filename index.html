🔍 Realations는 어떤 서비스인가요?
📊 소셜미디어 팔로워 관계 분석
인스타그램·틱톡·X·쓰레드·링크드인의 팔로워 데이터를 분석하여 맞팔 여부, 팬, 차단 계정 등을 한눈에 확인할 수 있는 서비스입니다.
🔒 안전한 파일 업로드 방식
각 플랫폼의 공식 데이터 내보내기 ZIP/CSV 파일을 업로드하는 방식으로, 비밀번호 입력이 불필요하고 파일이 서버로 전송되지 않아 안전합니다.

• 나만 팔로우하는 팬 목록
• 차단·제한·뮤트 계정
• 팔로우 요청 대기 목록
